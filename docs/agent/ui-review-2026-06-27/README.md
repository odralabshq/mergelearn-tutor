## Read this first

This folder is a visual review packet for the current MergeLearn Tutor platform UI.

The goal is not to prove backend correctness. The goal is to judge the look, navigation, interaction model, page organization, and how the frontend should be reorganized into a more intuitive learning workbench.

## Deep Research launch files

- `prompts/01-ui-ux-deep-research.md` — the explicit prompt to paste into Deep Research.
- `agent-drag-drop-packets/01-ui-ux-deep-research/` — self-contained launch folder with prompt and numbered context files.
- `04-report-template.md` — required report shape for the external agent.
- `06-agent-launch-checklist.md` — shortest manual launch checklist.
- `outputs/` — save returned Deep Research reports here.

## Supporting evidence files

- `contact-sheet.png` — overview of all captured pages.
- `screenshot-index.tsv` — route, screenshot path, and dimensions for each page.
- `screenshots/` — full-page screenshots of every current browser page. Exact filenames are listed in `screenshot-index.tsv`.
- `page-functionality.md` — supporting notes for each page.
- `frontend-review-packet.md` — design critique and recommended reorganization.
- `00-current-state-brief.md` and `01-source-packet.md` — context and attachment list.
