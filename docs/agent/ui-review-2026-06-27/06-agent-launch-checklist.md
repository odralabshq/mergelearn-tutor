# Agent launch checklist

Use this checklist when launching the Deep Research run.

- [ ] Attach `prompts/01-ui-ux-deep-research.md` or `00-AGENT-PROMPT.md` from the drag-drop folder.
- [ ] Attach `00-current-state-brief.md`.
- [ ] Attach `page-functionality.md`.
- [ ] Attach `frontend-review-packet.md`.
- [ ] Attach `screenshot-index.tsv`.
- [ ] Attach `contact-sheet.png`.
- [ ] Attach all 11 page screenshots from `screenshots/`.
- [ ] Tell the agent to follow `04-report-template.md`.
- [ ] Save the completed report to `outputs/01-ui-ux-deep-research-report.md`.
- [ ] After the report returns, synthesize it into implementation tickets before coding.

Do not attach secrets, credentials, local config files, or unrelated repo files.
